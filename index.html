Grundsätzliche Voraussetzungen zum Ausführen der Beispiele

Die hier gezeigten Beispiele nutzen in einigen Fällen experimentelle CSS3 Web-Technologien. Diese sind unter Umständen auf älteren Browsern nicht lauffähig sind. Sollten Sie die Beispiele herunterladen und auf Ihrem System ausführen, sollten Sie diese innerhalb von einem HTTP-Server starten.
Falls sie die Ablaufumgebung NodeJS bereits installiert haben, empfehlen wir hierfür das Paket http-server. Dieser Ermöglicht es uns über die Konsole einen minimalen Webserver zu starten, der den aktuellen Ordner ausliefert.
Die benutzt Testumgebung Karma benötigt ebenfalls NodeJS. Falls Sie bereits NodeJS auf Ihrem System installiert habenm, können Sie die beiden Module über folgenden Befehl installieren. 

$ npm install -g http-server karma
